# Contributing to invoice-app

Thank you for your interest in contributing to this project! Please follow these steps to submit your contribution:

## 1. Fork the repository
Click "Fork" at the top right of the repository page to create a copy under your own GitHub account.

## 2. Clone your fork
```bash
git clone https://github.com/your-username/invoice-app.git
```

## 3. Create a branch for your feature or bugfix
```bash
git checkout -b your-feature-branch
```

## 4. Make your changes, commit, and push
```bash
git add .
git commit -m "Describe your changes"
git push origin your-feature-branch
```

## 5. Open a Pull Request
Go to your repository on GitHub and click "Compare & pull request".
Describe clearly what your contribution does (feature, fix, etc).

## 6. Review & merge
Your pull request will be reviewed before merging.
