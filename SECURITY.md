# Security Policy

## Supported Versions

The following table lists the versions of Invoice App that are currently receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |
| < 0.0.x | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not create a public issue**. Instead, follow these steps:

1. **Email**: Send a detailed report to **<diomankeita001@gmail.com>**.
2. **Details**: Include a description of the vulnerability, steps to reproduce, and any potential impact.
3. **Wait**: Please allow up to 48 hours for a response. We will keep you updated on the progress of the fix.

We take security seriously and appreciate your help in keeping this project safe for everyone.

## Obsolete Dependencies

We regularly audit our dependencies. If you notice an obsolete or insecure package (e.g., an outdated `npm` dependency), please feel free to open a Pull Request with the update.

Thank you!
